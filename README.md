<h2>Installation</h2>
<p>1. Clone the repo</p>

```bash
git clone https://github.com/aridanpantoja/alura-space.git
```
<p>2. Install packages</p>

```bash
npm install
```
<h2>Running</h2>

```bash
npm run dev
```
